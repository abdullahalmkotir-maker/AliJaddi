# تطبيق عقد (Euqid)



موقع التطبيق: `Applications AliJaddi/تطبيق عقد` داخل [alijadditechnology/AliJaddi](https://github.com/alijadditechnology/AliJaddi).



## التشغيل (المدخل المؤقت)



```bash

pip install -r requirements.txt

streamlit run main.py

```



## إكمال المشروع



1. شغّل `_move_euqid_here.ps1` لنسخ المشروع من سطح المكتب إن وُجد، أو انسخ ملفات **Euqid** الكاملة هنا.

2. حدّث `requirements.txt` و`main.py` حسب التطبيق الكامل.



**Manifest:** `addons/manifests/euqid.json`  

**السحابة:** [Supabase](https://nzevwjghbvrrmmshnvem.supabase.co) للمزامنة مع الحساب؛ **مجلد `AliJaddi Cloud`** في المستودع = هجرات SQL للمخطط فقط — **ليس** لتخزين نصوص العقود أو بيانات المستخدم على القرص.
