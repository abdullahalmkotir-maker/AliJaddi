# Hussein12

مكتبة بايثون عالية المستوى بواجهات برمجة تطبيقات **بالعربية** لبناء منصات تدمج الذكاء الاصطناعي، متجر التطبيقات، ونظام الإضافات.

## التثبيت

```bash
pip install -e .
```

لاعتماديات اختيارية لوحدة الذكاء الاصطناعي (BentoML، transformers، PyTorch):

```bash
pip install -e ".[ai]"
```

## التحقق السريع

```python
import hussein12

print(hussein12.مرحباً())
```

## سطر الأوامر (v0.1)

```bash
hussein12 بدء_مشروع اسم_المشروع
```

ينشئ مجلد مشروع بملف `main.py` أولي.

## خادم المنصة (FastAPI)

`بدء_المنصة(...).تشغيل(المضيف=..., المنفذ=8000)` يشغّل **uvicorn** مع واجهة **OpenAPI** (عناوين عربية في الوثائق). مسارات مثل `/صحة`، `GET /api/متجر/تطبيقات`، `POST /api/نماذج/{model_index}/تنبؤ` وجسم JSON `{"البيانات": ...}`. الوثائق التفاعلية: `/docs` و `/redoc`. للاختبار بدون حظر: `تشغيل(ملخص_فقط=True)`. يمكن أيضاً `from hussein12 import انشاء_تطبيق_ويب` وربط التطبيق يدوياً.
